How it works

How you get
your next card.

The sniper runs in the background and handles everything. You just tap Claim.

Step 01
Scan
Every 2 minutes, scans all client boards for new cards in Next Up.
Step 02
Filter
Skips health brands, internal boards, and cards without a script.
Step 03
Alert
Sends you a Slack DM with the card name, brand, and script preview.
Step 04
Claim
One tap on the Claim button in Slack. Done.
Step 05
Pause
Stops alerting while you're already working on a card.

Setup

Two ways to
run it.

Pick whichever suits you. Option 2 is better but takes a few extra steps.

Option 1 — Laptop

Runs on your computer

Works when your laptop is on. Starts automatically at login.

  1. Install Python 3.12 from the Microsoft Store
  2. Download this repo: github.com/videoaditor/Card-Sniper- → Code → Download ZIP → unzip
  3. Inside the folder, copy sniper_config.example.json → rename to sniper_config.json → open in Notepad and fill in:
    "editor_label": "yournameb"
    "slack_user_id": "your Slack member ID"
    "slack_token": "xoxb-4769942055110-11448408858656-99JUM03a4daDH5n0Ywyogdex"
    "trello_token": "generate below"
  4. Get your Slack member ID: click your name in Slack → ••• → Copy member ID
  5. Get your Trello token — open this link and click Allow:
    https://trello.com/1/authorize?expiration=never&scope=read&response_type=token&name=CardSniper&key=65440fd2bb89d722659e7aecfc7f5799
  6. Open a terminal in the folder and run:
    pip install pytest
    python -m pytest test_card_sniper_core.py -q
    All 30 tests should pass.
  7. Set up auto-start — paste this in PowerShell (update the path if you unzipped somewhere different):
    $action = New-ScheduledTaskAction `
      -Execute (Get-Command python).Source `
      -Argument '"C:\Card Sniper\card_sniper.py" --loop' `
      -WorkingDirectory "C:\Card Sniper"
    $trigger = New-ScheduledTaskTrigger -AtLogOn -User $env:USERNAME
    $settings = New-ScheduledTaskSettingsSet -AllowStartIfOnBatteries -DontStopIfGoingOnBatteries -StartWhenAvailable -RestartCount 5 -RestartInterval (New-TimeSpan -Minutes 2) -ExecutionTimeLimit (New-TimeSpan -Hours 0)
    Register-ScheduledTask -TaskName "Card Sniper" -Action $action -Trigger $trigger -Settings $settings -RunLevel Limited -Force
    Start-ScheduledTask -TaskName "Card Sniper"
Option 2 — Cloud 24/7

Runs even when laptop is off

Hosted on Railway. Free tier is enough. Never misses a card.

  1. Complete Option 1 steps 1–6 first (install, config, test)
  2. Create a free account at railway.app
  3. New project → GitHub Repository → videoaditor/Card-Sniper- → Deploy Now
  4. Click the worker → Variables → Raw Editor and paste:
    TRELLO_API_KEY=65440fd2bb89d722659e7aecfc7f5799
    TRELLO_TOKEN=your trello token
    SLACK_TOKEN=xoxb-4769942055110-11448408858656-99JUM03a4daDH5n0Ywyogdex
    SLACK_USER_ID=your slack member ID
    EDITOR_LABEL=yournameb
    EDITOR_KEY=yourname
    WEEKLY_CAP=4
  5. Click Update Variables → Deploy. Done — it runs 24/7.
